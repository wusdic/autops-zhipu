# AUTOPS 错误码

> 文档状态：current
> 建议路径：`docs/03-api/ERROR_CODES.md`

---

## 错误码格式

`{模块2位}{级别1位}{序号2位}`

- 模块：00=通用, 01=资产, 02=配置, 03=凭证, 04=采集, 05=状态, 06=事件, 07=告警, 08=策略, 09=自动化, 10=日志, 11=AIops, 12=知识, 13=工单, 14=治理
- 级别：1=参数错误, 2=业务错误, 3=权限错误, 4=系统错误

## 通用错误码 (00xxx)

| 错误码 | HTTP | 说明 |
|---|---|---|
| 00001 | 500 | 未知错误（通用异常处理器会记录完整 traceback 与 trace_id） |
| 00010 | 422 | 参数验证失败（`ValidationError`） |
| 00011 | 400 | 请求格式错误 |
| 00100 | 401 | 未认证（中间件拦截，缺少/无效 Token） |
| 00101 | 401 | Token 过期 |
| 00102 | 401 | Token 无效 |
| 00103 | 403 | 权限不足（含 `require_admin` 管理端点鉴权失败） |
| 00104 | 403 | 账户已锁定 |
| 00200 | 404 | 资源不存在 |
| 00201 | 409 | 资源已存在 |
| 00202 | 409 | 资源冲突 |
| 00300 | 500 | 数据库错误 |
| 00301 | 500 | 缓存错误 |
| 00400 | 502 | 外部服务错误 |
| 00401 | 503 | AI 服务不可用 |
