## 1. 全局设计原则

### 1.1 一致性
- **统计卡片**：每个列表页顶部统一 4-6 个统计卡片，展示关键指标
- **工具栏**：搜索 + 筛选下拉 + 操作按钮（新建/刷新/导出）
- **表格**：el-table + stripe + sortable columns + 行内操作按钮
- **分页**：el-pagination，默认 page_size=20，最大 100
- **弹窗**：新建/编辑用 el-dialog，详情用 el-drawer
- **状态标签**：统一使用 StatusBadge / SeverityBadge 组件

### 1.2 视觉层次
- 统计卡片用不同背景色区分（蓝/绿/橙/红/紫）
- 表格行状态用 el-tag 颜色区分
- 危险操作（删除）用红色确认弹窗
- 空状态展示 el-empty 组件

### 1.3 交互规范
- 加载状态用 v-loading
- 操作反馈用 ElMessage.success/error
- 批量操作需先勾选，工具栏显示已选数量
- 表格行支持双击查看详情
- 列表页支持 Ctrl+K 快捷搜索
