## 2. 核心工作流

### 2.1 资产纳管流程

```text
发现资源 → 选择纳管 → 填写信息 → 绑定凭证 → 绑定采集模板 → 确认
→ 自动开始采集 → 状态可见 → 可告警、可分析、可执行
```

### 2.2 告警处理流程

```text
告警列表（按严重度排序）
  → 查看告警详情（上下文：资产、指标、日志、事件）
  → AI 分析（可选）
  → 确认告警
  → 选择处置方式：
    a. 手动处置 → 转自动化执行
    b. 策略自动处置 → 查看策略命中
    c. 转工单 → 指派处理
    d. 抑制/静默
  → 验证结果
  → 关闭告警
```

### 2.3 故障处置流程

```text
进入故障处置台
  → 左侧：时间线（事件按时间排列）
  → 中间：证据与分析（指标、日志、配置变更、AI 分析）
  → 右侧：推荐动作（策略匹配、AI 推荐）
  → 底部：实时日志流
  → 选择动作 → dry-run（可选）→ 审批（如需）→ 执行
  → 实时查看执行日志
  → 验证结果
  → 成功关闭 / 失败回滚 / 转工单
```

### 2.4 策略编排流程

```text
选择触发条件（事件/告警类型）
  → 定义适用范围（资产过滤）
  → 设计动作链（脚本/Playbook）
  → 设置风险等级和审批要求
  → 模拟测试（dry-run）
  → 发布策略
  → 监控命中和执行情况
```

### 2.5 知识沉淀流程

```text
工单关闭 / 告警解决
  → 系统提示生成知识草稿
  → AI 自动生成草稿（含处置步骤）
  → 人工审核修改
  → 关联策略和 Playbook
  → 发布知识
  → 后续相似告警可命中此知识
```

## 3. 交互模式

### 3.1 全局导航

- 顶部：平台 Logo、搜索框、通知铃铛、用户菜单
- 左侧：七大工作台菜单（可折叠）
- 主区域：当前工作台内容

### 3.2 搜索

- 全局搜索：资产、告警、工单、知识
- 快捷键：Ctrl+K

### 3.3 快捷操作

- 告警列表：行内快速确认、转工单
- 资产列表：行内快速采集、查看状态
- 执行列表：行内查看日志、取消

### 3.4 批量操作

- 资产：批量导入、批量绑定凭证、批量分组
- 告警：批量确认、批量抑制
- 执行：批量取消

## 4. 数据可视化

### 4.1 运维指挥台仪表盘

- 平台健康概览（组件状态灯）
- 严重告警数（实时）
- 今日事件数/告警数/工单数
- 采集成功率（24h 趋势线）
- 自动化执行统计（成功率、今日执行数）
- 待审批动作
- Top 10 告警资产

### 4.2 监控面板

- 资产状态分布（饼图：健康/告警/离线）
- 指标趋势（折线图，可叠加多资产）
- 告警热力图（按时间×资产）
- 事件流（实时滚动列表）
