# ADR 模板

> 本目录存放 AUTOPS 架构决策记录（Architecture Decision Records）

## 命名规范

`ADR-{编号}-{标题}.md`，例如 `ADR-0001-mysql-selection.md`

## 模板

```markdown
# ADR-{编号} {标题}

## 状态
proposed / accepted / superseded

## 背景
（描述问题上下文）

## 决策
（描述做出的决策）

## 备选方案
（列出考虑过的其他方案）

## 影响
（描述决策带来的影响）

## 后续动作
（描述需要的后续步骤）
```
